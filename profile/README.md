# M1A5TO ‚Äì 15-Minute City Housing Finder

**Short description**  
The goal of the project is to build a web application which, based on user-provided input, searches for apartments across Poland and analyzes them in the context of the 15-minute city concept, additionally determining their key parameters (price, size, accessibility to POI, profile-based score).
